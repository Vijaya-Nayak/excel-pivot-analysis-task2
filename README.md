
# Excel Pivot Table Analysis – Task 2 (Superstore Sales Dataset)

##  Objective
To analyze the Superstore Sales dataset using Excel Pivot Tables and generate meaningful business insights through summary analysis and visualizations.


##  Dataset
- **Superstore Sales Dataset**
- Source: Kaggle  
- The dataset contains information about orders, sales, profit, category, segment, region, and customer details.


##  Tools Used
- Microsoft Excel  
  

##  Work Performed
- Verified and cleaned the dataset for missing values, incorrect formats, and inconsistent categories
- Converted the dataset into an Excel table for dynamic analysis
- Created Pivot Tables for:
  - Total Sales by Category
  - Sales by Region and Segment
  - Profit Margin analysis by Category and Region
- Applied sorting and conditional formatting to highlight top-performing segments
- Added slicers (Region, Category, Segment) for interactive filtering
- Created Pivot Charts to visualize sales distribution
- Documented key business insights
- Exported the pivot analysis report as a PDF



##  Deliverables
- **Pivot_Report.xlsx** – Excel file containing pivot tables, slicers, and charts  
- **Pivot_Report.pdf** – Exported pivot analysis report  
- **Insights.docx / Insights.txt** – Document containing summarized business insights  



##  Final Outcome
Successfully analyzed the Superstore Sales dataset using Excel Pivot Tables and produced an interactive, insight-driven report demonstrating practical data analysis and storytelling skills.
